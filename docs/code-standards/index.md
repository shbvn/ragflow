RAGFlow maintains consistent, high-quality code across Python, Go, TypeScript/React, and C++ through documented standards and best practices.

## Quick Reference

### File Sizes
- **Target:** <200 LOC per file
- **Maximum:** 400 LOC (split larger files)
- **Exceptions:** Data models, autogenerated code

### Type Hints
**Mandatory for all languages:**
- Function signatures (inputs + returns)
- Class attributes
- Async function markers

### Error Handling
- No silent failures or bare `except:`
- Log all errors with context
- Use proper exception types
- Re-raise or handle explicitly

### Architecture Principles
- **YAGNI:** You Aren't Gonna Need It
- **KISS:** Keep It Simple, Stupid
- **DRY:** Don't Repeat Yourself
- **SOLID:** Single responsibility, Open/closed, etc.
