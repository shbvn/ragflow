# Python Code Standards

## File Organization

**Naming Convention:**
- Module files: `snake_case.py`
- Class files: Single class per file when >200 LOC, named after class
- Utility files: `utility_name.py` (e.g., `rag_tokenizer.py`, `search.py`)
- Package directories: `snake_case/`

**File Size Limits:**
- Target: <200 LOC per file
- Maximum: 400 LOC (split larger files)
- Exception: Data models, autogenerated code

**Structure Template:**
```python
"""Module docstring: brief description."""

# Standard library imports
import os
from typing import Optional, List

# Third-party imports
from quart import Blueprint, request

# Local imports
from api.db.services import DocumentService
from rag.nlp.search import Dealer

# Constants
DEFAULT_CHUNK_SIZE = 2048

# Functions/Classes
def my_function() -> str:
    """Docstring with type hints."""
    pass
```

## Type Hints & Validation

**Mandatory:**
- Function signatures (input + return types)
- Class attributes (via `__init__` annotations)
- Async functions (use `async def`)

**Example:**
```python
from typing import Optional, List, Dict, AsyncGenerator
import beartype

@beartype.beartype
def search_documents(
    query: str,
    top_k: int = 10,
    threshold: float = 0.5

    """Search and return ranked documents."""
    pass

async def stream_response(
    user_id: str,
) -> AsyncGenerator[str, None]:
    """Stream response chunks."""
    yield "chunk"
```

## Error Handling

**Pattern:**
```python
class DocumentParseError(Exception):
    """Raised when document parsing fails."""
    pass

try:
    result = parse_document(file_path)
except DocumentParseError as e:
    logger.error(f"Failed to parse {file_path}: {e}")
    raise
except Exception as e:
    logger.error(f"Unexpected error: {e}")
    raise
```

**Rules:**
- No bare `except:` or silent failures
- Log all errors with context
- Re-raise or handle explicitly

## Async/Await Patterns

**For I/O operations (HTTP, database, file):**
```python
async def fetch_and_process(user_id: str) -> Dict:
    """Async pattern for multiple I/O ops."""
    # Sequential when dependent
    user = await db.get_user(user_id)
    docs = await db.get_documents(user_id)
    
    # Parallel when independent
    embeddings, metadata = await asyncio.gather(
        embed_chunks(docs),
        extract_metadata(docs)
    )
    
    return {"user": user, "embeddings": embeddings}
```

**Never block the event loop:**
- No `time.sleep()` in async functions
- No blocking I/O in async context
- Use `asyncio.sleep()` for delays

## Database Access (Peewee ORM)

**Pattern:**
```python
from api.db import db
from api.db.db_models import Document, Chunk

class DocumentService:
    @staticmethod
    def create_document(name: str, kb_id: str) -> Document:
        """Create document with validation."""
        with db.atomic():  # Transaction
            doc = Document.create(
                name=name,
                kb_id=kb_id,
                status="PARSING"
            )
        return doc
    
    @staticmethod
    async def batch_update_status(
        doc_ids: List[str],
        status: str
    ) -> int:
        """Batch update with atomic transaction."""
        with db.atomic():
            query = Document.update(status=status).where(
                Document.id.in_(doc_ids)
            )
            return query.execute()
```

**Best Practices:**
- Use transactions (`with db.atomic():`) for multi-step operations
- Batch operations when possible
- Index frequent query columns
- Avoid N+1 queries (use `.prefetch()`)

## Logging

**Setup in entry point:**
```python
import logging

logger = logging.getLogger(__name__)

logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s - %(name)s - %(levelname)s - %(message)s"
)
```

**Usage:**
```python
logger.info(f"Processing document {doc_id}")
logger.warning(f"High latency: {query_time_ms}ms")
logger.error(f"Failed to embed: {error}", exc_info=True)
logger.debug(f"Query: {query_text[:100]}...")
```

## Testing

**File placement:**
- `test/` directory mirrors `api/`, `rag/`, `agent/` structure
- Test file: `test_<module_name>.py`

**Pattern:**
```python
import pytest
from api.apps.services import DocumentService

@pytest.fixture
def sample_document():
    """Setup test data."""
    return {"name": "test.pdf", "kb_id": "kb-1"}

@pytest.mark.p1
def test_document_creation(sample_document):
    """Test successful document creation."""
    doc = DocumentService.create_document(**sample_document)
    assert doc.name == sample_document["name"]
    assert doc.status == "PARSING"

@pytest.mark.p2
async def test_batch_status_update():
    """Test async batch operation."""
    doc_ids = ["doc-1", "doc-2"]
    count = await DocumentService.batch_update_status(doc_ids, "DONE")
    assert count == 2
```
