# Architecture Decision Records (ADRs)

Record significant decisions as short, immutable documents. Each ADR is a file
named `NNNN-short-title.md` where NNNN is a zero-padded sequence number.

## Template

```markdown
# ADR-NNNN: <Title>

**Status:** Proposed | Accepted | Deprecated | Superseded by ADR-NNNN
**Date:** YYYY-MM-DD
**Deciders:** <names>

## Context
What is the issue? What forces are at play?

## Decision
What did we decide?

## Consequences
What becomes easier? What becomes harder?

## Alternatives considered
Brief description of other options and why they were rejected.
```
